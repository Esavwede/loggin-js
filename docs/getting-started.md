## Quick Start
* Install with npm
```bash
npm install loggin-js --save
```

* Test it works, and see how it looks:
```bash
node run examples/basic-example.js
```

## Usage
### Using in node
```javascript
// Require the logging library
const logging = require('loggin-js');
```
